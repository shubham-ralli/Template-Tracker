Easily filter and export posts by template and status for streamlined content management and analysis.

== Description ==

Template Tracker is a WordPress plugin that provides a straightforward way to view and manage posts based on the template they use and their current status (published, draft, private, or trashed). This tool is ideal for quickly tracking the templates used in various posts and exporting this data for analysis.

= Features =
* Filter posts by templates used.
* Filter by post status (Published, Draft, Private, Trash).
* View post titles, templates, and statuses in a table.
* Download the filtered list in CSV format.

== Installation ==

1. Download the plugin and upload it to your WordPress plugin directory, or install directly through the WordPress plugins menu by clicking on "Add New" and searching for "Template Tracker."
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to `Tools > Template Tracker` to use the plugin.

== Frequently Asked Questions ==

= Can I filter by custom post types? =
Yes, Template Tracker allows filtering across multiple post types.

= What post statuses can I filter by? =
You can filter by Published, Draft, Private, and Trash statuses.

= Can I download the results? =
Yes, after filtering, you can download the results as a CSV file.
